
# Discord Meme Rate Bot

**Meme Rate Bot** هو بوت لديسكورد يراقب قنوات الميمز، يضيف تفاعلات تقييم تلقائيًا، ويقوم بحذف الميمز التي تحصل على تقييم سلبي أعلى بعد انتهاء فترة التصويت.



## المتطلبات

* Node.js 18 أو أعلى
* إنشاء تطبيق وبوت من [بوابة ديسكورد للمطورين](https://discord.com/developers/applications)
* تفعيل **MESSAGE CONTENT INTENT** في صفحة إعدادات البوت (للسماح بمراقبة الرسائل)



## الإعداد

1. تعديل القيم في ملف `.env`:

```
DISCORD_TOKEN=توكن البوت
CLIENT_ID=معرف التطبيق / Client ID
```

2. تثبيت الحزم:

```bash
npm install
```

3. تسجيل أوامر الـ Slash:

```bash
npm run register
```

4. تشغيل البوت:

```bash
npm run start
```



## أوامر الإدارة (Slash Commands)

* `/memerate status` : عرض الإعدادات الحالية
* `/memerate setduration minutes:<number>` : تحديد مدة التصويت بالدقائق
* `/memerate setemojis positive:<emoji> negative:<emoji>` : تحديد إيموجيات التصويت
* `/memerate addchannel channel:<#channel>` : إضافة قناة للمراقبة
* `/memerate removechannel channel:<#channel>` : إزالة قناة من المراقبة



## طريقة العمل

1. عند إرسال رسالة تحتوي على **صورة أو فيديو** في قناة مفعّلة:

   * يضيف البوت تلقائيًا تفاعلين (إيجابي/سلبي).
   * يبدأ مؤقت حسب مدة الإعداد.

2. عند انتهاء المؤقت:

   * يجمع البوت عدد التفاعلات لكل إيموجي (باستثناء تفاعل البوت نفسه).
   * إذا كان عدد التفاعلات السلبية أكبر من الإيجابية، يتم حذف الرسالة.
   * إذا كانت الإيجابية مساوية أو أكبر، تبقى الرسالة.
